## 🧵 What openstitch does

openstitch helps you draw web user interfaces with local LLMs on your own PC. You can use it to turn ideas into screen mockups, page layouts, and UI drafts without sending your work to an online service.

It is built for people who want to sketch web screens fast. You can use it to plan a dashboard, a landing page, a form, or a product view before you build the real app.

## 🖥️ What you need

openstitch works best on a Windows PC with:

- Windows 10 or Windows 11
- At least 8 GB of RAM
- A modern CPU
- 2 GB of free disk space
- A local LLM app or model setup if you want AI features to work on your machine

For smooth use, 16 GB of RAM gives better results. If your PC has a strong GPU, the app can feel faster, but it should still work on a normal home computer.

## 📦 How to set up local AI

openstitch uses local LLMs, so the AI runs on your machine. That keeps your work on your PC.

A simple setup looks like this:

1. Install a local LLM tool on Windows.
2. Download a model that fits your hardware.
3. Start the local model server or app.
4. Open openstitch.
5. Point openstitch to your local model if needed.

If you already use a local model tool, you can likely use the same setup here.

## 🛠️ Basic use

1. Open openstitch.
2. Create a new UI draft.
3. Type what you want to build.
4. Ask for a layout, a screen section, or a page style.
5. Move pieces until the design looks right.
6. Save your work when you finish.

A good prompt can be short and direct. For example:

- Create a clean dashboard for a fitness app
- Make a login page with email and password fields
- Build a pricing page with three plan cards
- Design a modern settings screen for a web app

## 🧭 Best results

Use clear words when you ask for a design. Say what screen you need, what parts belong on it, and what the page should help the user do.

Good examples:

- Create a product page with a hero area, features list, and call-to-action section
- Make a support page with search, tickets, and contact options
- Design a compact admin panel with a sidebar and stats cards

If the first result is not right, change your request and try again. Small changes can improve the layout a lot.

## 🔧 Common Windows issues

If the app does not open, try these steps:

- Make sure you downloaded the right file from the release page
- Unzip the file if the download came in a `.zip`
- Run the app again
- Check that Windows did not block the file
- Restart your PC and try once more

If the app opens but does not connect to the local model, check that your model tool is running and that the address is set right.

## 🔍 If the app feels slow

openstitch depends on your local AI setup, so speed can vary.

To improve speed:

- Close other heavy apps
- Use a smaller model
- Make sure your PC has enough free RAM
- Keep your model tool open while you work
- Use a faster disk if your model loads from storage often
